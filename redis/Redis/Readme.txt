- Create Redis server on local
	* Download redis zip file from as following link
		https://github.com/microsoftarchive/redis/releases
	* Run the redis-server.exe
	* Run the redis-cli.exe	
- Install redis
	pip install redis
- Install Redis GUI
	* Download zip file
		http://joeferner.github.io/redis-commander/
	* Go to extracted zip file path and open command prompt, then run as following commands
		npm install -g redis-commander
		redis-commander

- Run the async_sql_redis.py
	Read carefully discription on the top of code
